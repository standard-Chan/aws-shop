단, 구현하다가 아래 경우에 해당하면 코드 작성을 멈추고 나에게 먼저 알려주세요:
- 메서드명/클래스명이 도메인 언어와 어색하게 느껴질 때
- 테스트의 반환값/예외 설계가 구현상 맞지 않을 때
  이유를 설명하면 제가 수정 여부를 판단합니다.
